# monitoring-magang-dinkominfo
Internship Project (Dinas Komunikasi dan Informatika Kab. Pekalongan)
"Monitoring Magang"

# Roles
Superadmin
- Menambahkan divisi baru
- Mengubbah nama divisi
- Membuat akun pembimbing baru
- Menambahkan Perguruan tinggi / Sekolah baru
- Mengubah pembimbing peserta magang
- Mengubah divisi peserta magang
- Mengubah divisi pembimbing

Pembimbing
- Menerima/menolak permohonan magang
- Melihat presensi peserta magang
- Menyetujui/menolak laporan harian peserta magang
- Memberi tugas kepada peserta magang
- Menyetujui/menolak pengajuan izin peserta magang
- Memberi penilaian kepada peserta magang
- Mengunduh penilaian peserta magang
- Mengunduh semua laporan peserta magang

Peserta
- Melakukan presensi
- Menambahkan laporan harian
- Menambahkan tugas baru
- Mengubah status tugas (menjadi selesai)
- Menambahkan pengajuan izin
- Melihat penilaian
- Mengunduh penilaian
- Mengunduh semua laporan
